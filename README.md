# Arduino-EBIMU_9DOFV2-Library
Arduino Library for E2BOX EBIMU-9DOFV2 AHRS Module
* getEulerAngles()
* getQuaternion()
* getGyro()

# Warnings
You shoud change serial pin configuration, which is at the top of EBIMU.h header, to fit to your serial configuration.
